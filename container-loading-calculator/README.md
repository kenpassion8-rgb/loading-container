# Container Loading Calculator - 货柜装箱计算器

## 简介

这是一个 Claude Code Skill，用于帮助用户计算货柜（集装箱）的最大装箱数量。支持20尺平柜（20GP）和40尺高柜（40HQ）两种标准货柜类型。

## 功能特点

- ✅ 自动计算最优装箱方案
- ✅ 支持20尺平柜（默认）和40尺高柜
- ✅ 自动添加1cm预留空间
- ✅ 考虑重量和体积双重限制
- ✅ 预留关门和安全间隙空间
- ✅ 支持指定摆放方向

## 使用方法

### 触发方式

在 Claude Code 中，使用以下任一方式触发：

```
"计算货柜能装多少箱子"
"帮我算一下装箱数量"
"20尺柜能装多少"
"40尺高柜能装多少"
```

### 输入信息

触发后，向 Claude 提供以下信息：

1. **箱子尺寸**：长 × 宽 × 高（单位：cm）
2. **单箱重量**：kg
3. **货柜类型**（可选）：20GP 或 40HQ（不指定则默认20GP）
4. **摆放限制**（可选）：如"必须用 33.2×22 这一面朝下"

### 示例对话

**示例1：基本使用**
```
用户：帮我计算一下装箱数量
Claude：请提供以下信息：
1. 箱子的长、宽、高（cm）
2. 单箱重量（kg）
3. 使用什么货柜？（20尺平柜/40尺高柜，默认20尺）

用户：箱子 33.2*22*25.4cm，重量2.6公斤，20尺柜
Claude：[计算结果...]
```

**示例2：指定摆放方向**
```
用户：计算货柜装载，箱子33.2*22*25.4cm，2.6kg，必须用33.2*22这一面朝下
Claude：[计算结果...]
```

**示例3：40尺高柜**
```
用户：40尺高柜能装多少？箱子50*40*30cm，重量5kg
Claude：[计算结果...]
```

## 货柜参数

| 参数 | 20尺平柜 (20GP) | 40尺高柜 (40HQ) |
|------|-----------------|-----------------|
| 可用长度 | 569 cm | 1183 cm |
| 可用宽度 | 235 cm | 235 cm |
| 可用高度 | 234 cm | 264 cm |
| 配货毛重 | 21,000 kg | 26,000 kg |
| 最大容积 | 26 m³ | 76 m³ |

> 注：长度已预留20cm关门空间，高度已预留5cm安全间隙

## 计算规则

1. **尺寸预处理**：用户提供的尺寸每个维度自动 +1cm
2. **重量限制**：总重量 ≤ 货柜配货毛重
3. **体积限制**：总体积 ≤ 货柜最大容积
4. **最终箱数**：取重量限制和体积限制中的较小值

## 安装

将此文件夹复制到 Claude Code 的 skills 目录：

```bash
# Windows
xcopy /E /I container-loading-calculator %USERPROFILE%\.claude\skills\

# Mac/Linux
cp -r container-loading-calculator ~/.claude/skills/
```

## 注意事项

- 所有尺寸默认单位为厘米（cm）
- 重量默认单位为公斤（kg）
- 计算结果已包含预留空间，实际装载可能略有差异
- 复杂形状货物请咨询专业装箱人员
