# Container Loading Calculator - 货柜装箱计算器

## 简介

这是一个 Claude Code Skill，用于帮助用户计算货柜（集装箱）的最大装箱数量。支持20尺平柜（20GP）和40尺高柜（40HQ）两种标准货柜类型。

## 功能特点

- ✅ 自动计算最优装箱方案
- ✅ 支持20尺平柜（默认）和40尺高柜
- ✅ 排列计算时自动+1cm预留空间
- ✅ **体积计算使用原始尺寸**（关键规则）
- ✅ 同时考虑空间、重量、体积三重限制
- ✅ 顶部预留5cm安全间隙
- ✅ 返回满足所有限制的最大箱数

## 关键计算规则

### 1. 排列计算（使用+1cm尺寸）
- 排列长度 = 原始长度 + 1cm
- 排列宽度 = 原始宽度 + 1cm
- 排列高度 = 原始高度 + 1cm

### 2. 体积计算（使用原始尺寸）⚠️
```
单箱体积(CBM) = 原始长 × 原始宽 × 原始高 ÷ 1,000,000
```
**重要：体积计算必须使用原始尺寸，不能加1cm！**

### 3. 三重限制
| 限制类型 | 计算方式 |
|---------|---------|
| 空间限制 | 每层箱数 × 层数 |
| 重量限制 | 配货毛重 ÷ 单箱重量 |
| 体积限制 | 最大容积 ÷ 单箱体积 |

### 4. 最终箱数
```
最大装箱数 = min(空间限制, 重量限制, 体积限制)
```

## 货柜参数

| 参数 | 20尺平柜 (20GP) | 40尺高柜 (40HQ) |
|------|-----------------|-----------------|
| 可用长度 | 589 cm | 1,203 cm |
| 可用宽度 | 234 cm | 235 cm |
| 可用高度 | 233 cm | 264 cm |
| 配货毛重 | 17,500 kg | 22,000 kg |
| 最大容积 | 25 CBM | 65 CBM |

> 注：高度已预留5cm安全间隙

## 使用方法

### 触发方式

在 Claude Code 中，使用以下任一方式触发：

```
"计算货柜能装多少箱子"
"帮我算一下装箱数量"
"20尺柜能装多少"
"40尺高柜能装多少"
```

### 输入信息

触发后，向 Claude 提供以下信息：

1. **箱子尺寸**：长 × 宽 × 高（单位：cm）
2. **单箱重量**：kg
3. **货柜类型**（可选）：20GP 或 40HQ（不指定则默认20GP）
4. **摆放限制**（可选）：如"必须用 33.2×22 这一面朝下"

**示例：计算装箱数量**
```

2. 单箱重量（kg）
3. 使用什么货柜？（20尺平柜/40尺高柜，默认20尺）

用户：箱子 33.2*22*25.4cm，重量2.6公斤，20尺柜

- 单箱体积：33.2×22×25.4÷1,000,000 = 0.01855 CBM（按原始尺寸）

---

沿长度方向（589 cm）：589 ÷ 34.2 = 17 箱
沿宽度方向（234 cm）：234 ÷ 23 = 10 箱
沿高度方向（233 cm）：233 ÷ 26.4 = 8 层

每层箱数 = 17 × 10 = 170 箱

**最大装箱数 = min(1,360, 6,730, 1,347) = 1,347 箱**
```

## 安装

将此文件夹复制到 Claude Code 的 skills 目录：

```bash
# Windows
xcopy /E /I container-loading-calculator %USERPROFILE%\.claude\skills\

# Mac/Linux
cp -r container-loading-calculator ~/.claude/skills/
```

## 注意事项

- 所有尺寸默认单位为厘米（cm）
- 重量默认单位为公斤（kg）
- **体积计算必须使用原始尺寸，不能加1cm**
- 排列计算使用+1cm预留空间
- 最终箱数同时满足空间、重量、体积三重限制
- 复杂形状货物请咨询专业装箱人员
