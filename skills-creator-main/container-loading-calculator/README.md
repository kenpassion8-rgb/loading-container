# Container Loading Calculator - 货柜装箱计算器

## 简介

这是一个 Claude Code Skill，用于帮助用户计算货柜（集装箱）的最大装箱数量。支持20尺平柜（20GP）和40尺高柜（40HQ）两种标准货柜类型。

## 功能特点

- ✅ 自动计算最优装箱方案
- ✅ 支持20尺平柜（默认）和40尺高柜
- ✅ 自动添加1cm预留空间
- ✅ **箱子必须以长×宽为底面放置**
- ✅ 同时考虑空间、重量、体积三重限制
- ✅ 预留关门和安全间隙空间
- ✅ 返回满足所有限制的最大箱数

## 使用方法

### 触发方式

在 Claude Code 中，使用以下任一方式触发：

```
"计算货柜能装多少箱子"
"帮我算一下装箱数量"
"20尺柜能装多少"
"40尺高柜能装多少"
```

### 输入信息

触发后，向 Claude 提供以下信息：

1. **箱子尺寸**：长 × 宽 × 高（单位：cm）
2. **单箱重量**：kg
3. **货柜类型**（可选）：20GP 或 40HQ（不指定则默认20GP）

> **注意**：箱子将以 **长×宽为底面** 放置，高度方向堆叠。不支持侧放。

**示例：计算装箱数量**
```

用户：箱子 33.2*22*25.4cm，重量2.6公斤，20尺柜

## 📦 货柜装载计算结果
```

## 摆放规则

**箱子必须以长×宽为底面放置**

```
┌─────────────────┐
│                 │  ← 长×宽 底面
│                 │
└─────────────────┘
         ↑
      高度方向堆叠
```

禁止的摆放方式：
- ❌ 宽×高为底面（侧放）
- ❌ 长×高为底面（侧放）

## 计算逻辑

### 1. 尺寸预处理

**排列计算尺寸**（+1cm预留空间）：
- 排列尺寸 = 原始尺寸 + 1cm

**实际体积计算**（按原始尺寸）：
- 单箱体积 = 原始长 × 原始宽 × 原始高 ÷ 1,000,000

### 2. 排列计算（长×宽为底面）

**沿长度方向：**
```
569 cm ÷ (长+1) cm = ? 箱（余 ? cm）
```

**沿宽度方向：**
```
234 cm ÷ (宽+1) cm = ? 箱（余 ? cm）
```

**沿高度方向：**
```
233 cm ÷ (高+1) cm = ? 层（余 ? cm）
```

**每层箱数** = 长方向箱数 × 宽方向箱数

### 3. 三重限制计算

| 限制类型 | 计算公式 |
|---------|---------|
| 空间限制 | 每层箱数 × 层数 |
| 重量限制 | 配货毛重 ÷ 单箱重量 |
| 体积限制 | 最大容积 ÷ 单箱体积 |

### 4. 最终最大箱数
```
最大装载箱数 = min(空间限制, 重量限制, 体积限制)
```

### 5. 输出结果
- **最大装箱数**
- **柜子排列方式**：长×宽×高
- **总重量**
- **总体积**

## 安装

将此文件夹复制到 Claude Code 的 skills 目录：

```bash
# Windows
xcopy /E /I container-loading-calculator %USERPROFILE%\.claude\skills\

# Mac/Linux
cp -r container-loading-calculator ~/.claude/skills/
```

## 注意事项

- 所有尺寸默认单位为厘米（cm）
- 重量默认单位为公斤（kg）
- **箱子必须以长×宽为底面放置**，不支持侧放
- 计算结果已包含预留空间，实际装载可能略有差异
- 最终箱数同时满足空间、重量、体积三重限制
- 复杂形状货物请咨询专业装箱人员
